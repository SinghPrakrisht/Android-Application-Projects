# 🎵 Android Media Player App

## 📱 Overview
This Android application is a simple and functional media player that allows users to:
- Play audio files from local storage
- Stream video content from a URL
- Control playback using intuitive buttons

The app is designed with a clean UI and supports dark themes.

---

## ✨ Features

- 🎧 Play audio files from device storage
- 🌐 Stream video from a URL
- ⏯️ Playback controls:
  - Play
  - Pause
  - Stop
  - Restart
- 📂 Open local media files
- 🔗 Open media via URL
- 🌙 Dark theme support

---

## 🛠️ Tech Stack

- Java / Kotlin
- Android SDK
- XML for UI design
- MediaPlayer API
- VideoView / Streaming support

## 🎮 Controls

- **Open File** → Select audio file from device  
- **Open URL** → Enter video streaming URL  
- **Play** → Start playback  
- **Pause** → Pause media  
- **Stop** → Stop playback  
- **Restart** → Restart media from beginning
