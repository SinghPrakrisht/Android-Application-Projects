# Mobile Application Development (CSE3709) Assignment

This repository contains four Android application projects developed as part of the Mobile Application Development course assignment.

## Projects Included

### Q1: Currency Converter App
- Converts between INR, USD, EUR, and JPY
- Includes Light/Dark theme switching

### Q2: Media Player App
- Plays audio files from device storage
- Streams video from a given URL
- Includes controls: Play, Pause, Stop, Restart

### Q3: Sensors App
- Displays real-time data from:
  - Accelerometer
  - Light Sensor
  - Proximity Sensor

### Q4: Photo Gallery App
- Capture photos using device camera
- Save images to selected folder
- View images in grid layout
- View image details (name, path, size, date)
- Delete images with confirmation dialog

## Note
Each project is implemented as a separate Android Studio project and organized into individual folders as per assignment guidelines.
